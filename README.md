
## Description

This is workers-api services enables for the moments.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# first run the database in docker compose (more details: .env & docker-compose.yml)
$ docker-compose up -d <base-name>

# development
$ npm run start:dev

# production mode
$ npm run start:prod
```
